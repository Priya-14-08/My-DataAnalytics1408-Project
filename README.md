# Cost of Living Analysis & Prediction using Machine Learning

## Project Overview
This project analyzes and predicts the cost of living using machine learning techniques. It classifies living conditions as Affordable, Neutral, or Expensive.

## Technologies Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Features
- Cost of living prediction
- Multiple ML model comparison
- Clustering analysis
- Data visualization

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- SVM

## How to Run
1. Install required libraries:
   pip install pandas numpy scikit-learn matplotlib seaborn

2. Run the notebook:
   jupyter notebook

## Results
Random Forest gave the best performance and expense-to-income ratio is a key factor.
